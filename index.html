Emacs Starter Kit for the Social Sciences

This is a fork of Phil Hagelberg's emacs starter kit with additional tools included for social scientists. The added packages include up-to-date versions of ESS and ECB, amongst others. See the original README file for installation instructions. Essentially, the starter kit is meant to replace the contents of your ~/.emacs.d/ directory, if one exists. Back up your .emacs.d/ directory, put the starter kit in its place, either by unzipping it there or (if you have git installed) cloning the repository. On Linux and Mac OS X you do this: 

$ git clone git://github.com/kjhealy/emacs-starter-kit ~/.emacs.d
On Windows you must copy the directory to Documents and Settings/%your user name%/Application Data/

Then start emacs. To activate the "social science" customizations in this package, you will need to change the name of ~/.emacs.d/kjhealy.el that of your own username or the name of your computer. To find the value of these variables inside emacs, do 

C-h-v user-login-name
C-h-v system-name
Finally, inside the kjhealy.el, change the paths to any BibTeX databases as described at the top of the file.
